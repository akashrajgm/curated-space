# Curated Space - Architecture Level MVP

This is an elite, zero-latency digital environment parsing high-end structural nodes explicitly via ultra-minimalist, brutalist logic streams.

## Awwwards Tier Technical Implementations:

*   **The Tactile Fluid Scanner:** Completely dismantled standard vanilla cursors. Mapped a `6px` organic comet element using `framer-motion/useVelocity` to calculate real-time vectors tracking trajectory speed and snapping cleanly into an X-Ray inversion lens (`mixBlendMode: 'difference'`). Heavy elastic mass configurations (`stiffness: 250`).
*   **Tactile UI Soundscape:** Integrated `howler.js` dropping direct low-frequency pulses bridging the gap between tactile web-actions and audible responses utilizing `Mixkit.co` high-fidelity snaps.
*   **The GSAP Cinematic Scroll Engine:** Wired global ScrollTrigger algorithms directly traversing `<Canvas>` instances intercepting scale loops.
*   **Liquid Scale-Blur Architecture & Blur-Up Loading:** Replaced generic fades with deep `backdrop-filter: blur()`. Dynamically maps ultra-low resolution `&q=10` Unsplash thumbnails resolving beautifully via Framer into heavy 4K structural photography preventing native popping.
*   **3D Selective Bloom Post-Processing:** Pushed raw Three.js nodes straight through `@react-three/postprocessing` dropping strict vignette layers and sub-surface luminance targeting cinematic focal boundaries.
*   **Adaptive Performance Context:** Secured logic completely terminating 3D geometry preventing computational drag, utilizing an explicit System Context bottom level switch.

## Architectural Tech Stack

*   **Build Pipeline:** Vite
*   **Core Architecture:** React Native/DOM
*   **Routing Network:** React Router DOM
*   **WebGL Engineering:** Three.js, React-Three-Fiber, React-Three-Drei, PostProcessing
*   **Motion Mathematics:** GSAP (ScrollTrigger Engine), Framer Motion (Liquid Physics/Springs)
*   **Sensory Mappings:** Howler.js

## Boot Sequence

```bash
npm install
npm run dev
```

To compile zero-footprint payloads routing back to Node environments:
```bash
npm run build
```

Ensure `.env` (`VITE_API_URL`) matches your target endpoints (`apiClient.js`). All logs and local metrics are stripped explicitly for production execution.
